# document_scanner

A simple document scanner using Python with a GUI.

---

## Usage

```
python doc_scanner_app.py --image=<input_image_path>
```

or

```
python doc_scanner_app.py
```

and use filemenu to open an image.

---

## Dependencies

- Python 2.7x
- Tkinter
- OpenCV
- Pillow
- NumPy
